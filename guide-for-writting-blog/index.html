Guide-for-writting-blog

May 31st, 2015 3:45 pm

  1. Open the termianl.

  2. Change into the blog directory, cd Projects/Blog/octopress.

  3. rake new_page[“page title”] to create a page; rake new_post[“post title”] to create a article. Both of them will show the markdown file path.
  4. Open the markdown file with Mou, edit it and save.
  5. rake preview and open http://localhost:4000 to view the effect on localhost.
  6. If without other problem, deploy it to github with rake deploy.
